# Инструкция по подготовке этикетки к печати (58x40 мм)

## Где открывать
- Откройте карточку SKU: `/sku/<id>/edit/`.
- Нажмите кнопку <Печать этикетки>.
- Настройки оборудования: `/labels/settings/`.

## Какие поля участвуют
Инфоблок (2-3 колонка):
- Наименование (берется из <Название для печати>, если пусто - из <Название>).
- Размер (показывается, если заполнен).
- Бренд (скрывается, если значение `Nobrand`).
- Предмет, Цвет, Состав (значения только если заполнены).
- Поставщик (значение с подписью <Поставщик: :>).

Коды:
- Штрихкод (берется из поля <ШК товара>).
- QR-код (формируется из штрихкода и внешнего ID, если есть).
- EAC фиксированным изображением в правом нижнем углу.

## Подготовка к печати
1. Заполните поля SKU, особенно: <Название/Название для печати>, <Размер>, <Бренд>, <Предмет>, <Цвет>, <Состав>, <Поставщик>, <ШК товара>.
2. Откройте <Печать этикетки> и убедитесь, что все нужные строки отображаются.
3. В диалоге печати:
   - Формат бумаги: 58x40 мм (товар).
   - Масштаб: 100% (без <Подогнать>).
   - Поля: 0 мм, без колонтитулов.
4. Отправьте на принтер этикеток.

## Размеры этикеток
- Товар: 58x40 мм.
- Короб: 58x60 мм.
- Паллет: 75x120 мм.
